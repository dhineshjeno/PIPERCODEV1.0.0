# 🛸 PiperCode Activity Tracker

**Track your coding activity inside VS Code and visualize your productivity!**

PiperCode is a gamified, high-performance coding activity tracker. It silently monitors your keystrokes, detects over 60 programming languages, and builds an accurate profile of your daily coding habits. 

Featuring a stunning **Neo-Brutalist dashboard**, PiperCode doesn't just track your time — it makes productivity fun.

## ✨ Features

- **Real-Time Activity Tracking**: Tracks your coding time down to the second, automatically pausing when you're idle.
- **Neo-Brutalist Dashboard**: A beautiful, stylized dashboard to view your progress.
- **Language Detection**: Automatically detects and logs over 60 different programming languages.
- **Global Leaderboard**: Sign in with GitHub to compete on the global developer leaderboard.
- **Friends System**: Search for other developers, add them as friends, and track their progress live!
- **Daily Streaks**: Build a daily coding streak by hitting your customizable minute threshold.
- **Status Broadcast**: Set a custom status message (like "Vibe Coding...") for your friends to see.
- **Local Tasks**: A built-in scratchpad for your daily to-do list.

## 🚀 How to Use

1. **Install** the extension.
2. **Open a project folder** in VS Code and start typing in any code file.
3. The **Status Bar** at the bottom right will change from "Ready" to your live coding time!
4. Press `Ctrl+Shift+P` and run **`PiperCode: Open Dashboard`** to view your stats.
5. *(Optional)* Click "Connect GitHub" in the dashboard to sync your stats to the cloud and join the leaderboard.

## ⚙️ Configuration

You can customize PiperCode via VS Code Settings (`Ctrl+,`):

- `pipercode.streakDailyThresholdMinutes`: The minimum number of minutes you must code in a day to increment your daily streak (Default: 30).
